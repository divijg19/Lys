# Camellia

Camellia is a community-driven skill sharing platform built using AWS for cloud infrastructure, the MERN stack (MongoDB, Express.js, React, Node.js) for the web application, and Go for backend microservices.

## Features

- User authentication and profiles
- Skill listing and discovery
- Real-time messaging and notifications
- Secure file uploads with AWS S3
- RESTful APIs with Go microservices

## Tech Stack

- **Frontend:** React
- **Backend:** Node.js, Express.js, Go
- **Database:** MongoDB
- **Cloud:** AWS (EC2, S3, Lambda)
